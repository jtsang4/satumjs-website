Satumjs

💫 面向多实例集成、功能可插拔的微前端框架

快速上手 →

微内核

支持中间件、插件机制，可以集成其他微前端框架，和做有意义的扩展。

开箱即用

只需要引入 @satumjs/core，配置微应用的激活规则后即可看到效果。

集成能力强

支持所有的外部应用无需任何适配代码，就能无缝地集成。

# 📦 安装

$ yarn add @satumjs/core  # or npm i @satumjs/core -S

# 🔨 使用

import { register, start } from '@satumjs/core';

// 注册微应用
register([
  {
    name: 'reactApp',
    entry: 'http://localhost:8000',
    rules: {
      rule: '/',
      container: '#mountNode',
    },
  },
]);

// 启动
start();

或者使用远程的微应用配置，可以直接启动：



 


import { start } from '@satumjs/core';
start({
  remoteUrl: 'https://remote.test/micro-app-configs.json',
});

更多示例请参考：https://github.com/satumjs/examples (opens new window)
