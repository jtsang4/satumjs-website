SatumJS

💫 面向多实例集成、功能可插拔的微前端框架

快速上手 →

微内核

支持中间件、插件机制，可以集成其他微前端框架，和做有意义的扩展。

开箱即用

只需要引入 @satumjs/core，配置微应用的激活规则后即可看到效果。

集成能力强

支持所有的外部应用无需任何适配代码，就能无缝地集成。

支持多实例

内置激活规则计算，可以同时激活多个实例，甚至是多条“微应用依赖链”。

共享机制灵活

UI 上支持布局/区块/组件/三方库共享，数据逻辑层面支持跨微应用共享。

无缝支持 Vite

官方提供的沙箱中间件，完整的沙箱能力无缝支持 Vite。

# 📦 安装

$ yarn add @satumjs/core  # or npm i @satumjs/core -S

# 🔨 使用

import { register, start } from "@satumjs/core";

// 注册微应用
register({
  name: "reactApp",
  entry: "http://localhost:8000",
  rules: { rule: "/", container: "#mountNode" },
});

// 启动
start();

或者使用远程的微应用配置，可以直接启动：



 

import { start } from "@satumjs/core";

start({ remoteUrl: "https://example.com/apps.json" });
