# 官方中间件

# @satumjs/simple-midwares

https://www.npmjs.com/package/@satumjs/simple-midwares (opens new window)

一些简单中间件的实现集合。包含了对缓存、沙箱、css 隔离、获取挂载 dom、路由协调器的实现。

# @satumjs/midware-single-spa

https://www.npmjs.com/package/@satumjs/midware-single-spa (opens new window)

对 single-spa 做了一层封装，可以利用其 url 变化，协调微应用加/卸载的能力。

# @satumjs/midware-thead-occ

https://www.npmjs.com/package/@satumjs/midware-thead-occ (opens new window)

针对 occ 网站业务，封装了一个中间件。除了借用 @satumjs/simple-midwares 的一些能力外，重点实现了浏览器 nosql 缓存机制、代码处理。
